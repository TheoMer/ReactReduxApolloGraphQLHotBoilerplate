## Installation

```bash
npm install
```

## Running dev server

You do **NOT** need to run your own GraphQL server. Simply run the `npm run dev` command. If you want to work with your own backend change the `API_URL` value in `src/config/index.js`.

## Running tests

```bash
npm run test
```
